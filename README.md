# 🚨 Gemma-Rescue Edge

Gemma-Rescue Edge is an offline AI-powered disaster response system built using Gemma, Unsloth fine-tuning, Ollama, and Streamlit.
The system converts thermal readings, movement signals, and environmental conditions into actionable rescue intelligence entirely offline.

The system analyzes:

* thermal readings
* victim movement
* environmental conditions

and generates:

* rescue priority
* medical risk assessment
* emergency response recommendations

## 🔥 Features

* Offline AI inference
* Quantized GGUF deployment
* Disaster triage reasoning
* Streamlit rescue dashboard
* Ollama local inference
* Disaster victim prioritization
* Emergency rescue recommendations
* Edge-AI deployment

## 🛠️ Tech Stack

* Gemma
* Unsloth
* Ollama
* Streamlit
* Python
* Hugging Face Datasets
* TRL / PEFT / LoRA
* GGUF Quantization

## 🚀 Run Locally

```bash
ollama run gemma-rescue-edge
```

```bash
python -m streamlit run app.py
```

## 📌 Project Goal

To assist disaster response teams with offline AI-powered rescue assessment during emergencies where internet connectivity may be unavailable.
